Section 02

Stochastic Transitions

When the agent chooses an action, the environment doesn't always comply. The intended action is simply whichever action is being evaluated — it succeeds with p = 0.8. With p = 0.2 the agent slips to one of the two perpendicular directions (0.1 each). The agent never moves backward. Hitting a wall keeps the agent in place.

Section 03

The Update Rule

Q-values are updated synchronously — every cell uses Q values from the previous iteration. R(s) is the reward at the current state, fixed regardless of outcome. The discounted expected future value is then added.

Qk+1(s, a) = R(s) + γ · E[ maxa' Qk(s') ] = R(s) + 0.9 · [0.8·maxQk(s'intended) + 0.1·maxQk(s'perp1) + 0.1·maxQk(s'perp2)] Dir(s) = argmaxa Q(s, a) — the greedy policy arrow shown in the grid

Because all Q₀ = 0, Iteration 1 gives Q₁(s,a) = R(s) for every action — no directional preference yet. Differentiation begins in Iteration 2 as values propagate from neighbours.
